Implement the feature or fix described. Follow incremental implementation:
1. Identify the vertical slice (smallest end-to-end working version)
2. Build the walking skeleton first (hardcoded, no edge cases)
3. Write tests for the skeleton
4. Flesh out the implementation incrementally
5. Add edge cases with tests
6. Leave the system in a deployable state at every commit

Do not implement the entire feature in one pass. Each increment must be testable and deployable.
